# Web Project Organizer

Página web desarrollada en **HTML y Bootstrap** que funciona como un **organizador o índice principal** para acceder a diferentes proyectos o ejemplos web dentro de un mismo sitio.

## Descripción

El proyecto consiste en una página principal (`index.html`) que contiene un **menú de navegación** con enlaces a distintas secciones o páginas web que muestran ejemplos de componentes y funcionalidades de Bootstrap.

## Funcionalidades

* Barra de navegación responsiva con Bootstrap.
* Acceso a múltiples páginas del proyecto.
* Organización de diferentes ejemplos web en un solo sitio.
* Visualización de imágenes responsivas.

## Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap 4
* JavaScript
* jQuery

## Secciones del proyecto

El menú permite acceder a diferentes páginas como:

* **SINBS** – Página sin Bootstrap.
* **CONBS** – Página con Bootstrap.
* **Fuentes** – Ejemplo de tipografías.
* **Formulario** – Formulario interactivo.
* **Grids** – Ejemplo del sistema de rejillas de Bootstrap.
* **ImgResponsiva** – Uso de imágenes responsivas.
* **Login** – Ejemplo de sistema de inicio de sesión.
